## [0.1.0] - 2024-10-07

### 新增
- ✨ **YButton 组件**: 支持多种类型、尺寸和状态的按钮组件
  - 支持 `primary`、`success`、`warning`、`error` 等类型
  - 支持 `small`、`medium`、`large` 三种尺寸
  - 支持禁用状态和加载状态
  - 支持图标按钮和文字按钮

- ✨ **YCard 组件**: 灵活的卡片容器组件
  - 支持标题、内容、底部操作区域
  - 支持阴影效果配置
  - 支持边框样式定制
  - 响应式设计，适配移动端

- ✨ **YInput 组件**: 功能丰富的输入框组件
  - 支持多种输入类型 (text, password, number 等)
  - 支持前缀、后缀图标
  - 支持清除按钮
  - 支持禁用和只读状态
  - 完整的表单验证支持

- ✨ **YTextarea 组件**: 多行文本输入组件
  - 支持自动调整高度
  - 支持字符计数显示
  - 支持最大长度限制
  - 支持禁用和只读状态

- ✨ **YCheckbox 组件**: 多选框组件
  - 支持单个复选框和复选框组
  - 支持全选/反选功能
  - 支持禁用状态
  - 支持自定义选中值

- ✨ **YRadio 组件**: 单选框组件
  - 支持单选框组
  - 支持禁用状态
  - 支持自定义选中值
  - 完整的键盘导航支持

- ✨ **YSwitch 组件**: 开关组件
  - 支持多种尺寸
  - 支持自定义开关文本
  - 支持禁用状态
  - 流畅的切换动画

- ✨ **YPicker 组件**: 联动选择器组件
  - 支持 1-3 级联动选择
  - 支持自定义数据字段映射
  - 支持多种交互方式 (点击、滚轮、触摸)
  - 完整的移动端适配
  - 支持自定义样式主题

### 基础设施
- 🏗️ **项目架构**: 建立 Monorepo 项目结构
  - `packages/you-ui-core`: 核心组件库

  - 使用示例和最佳实践
  - 安装和配置指南

### 样式系统

  - CSS 变量支持
  - 深色/浅色主题切换

  - 响应式断点系统

- 📱 **响应式设计**: 完整的移动端适配
  - 移动端优先的设计理念

## 版本说明

### 版本号规则
- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 发布周期
- **主版本**: 每年 1-2 次重大更新
- **次版本**: 每月 1-2 次功能更新
- **修订版本**: 根据需要随时发布修复

### 兼容性承诺
- 在同一主版本内保持 API 向下兼容
- 废弃的 API 会在下个主版本移除前提供至少一个版本的过渡期
- 重要变更会在文档中详细说明迁移方法
