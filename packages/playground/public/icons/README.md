## 使用说明

所有图标均为 SVG 格式，可以通过以下方式在 YIcon 组件中使用：

```vue
<!-- 使用图标名称（不含 .svg 后缀） -->
<YIcon name="home" :size="24" />
<YIcon name="search" :size="32" color="#007aff" />
```

或者直接引用 SVG 文件：

```vue
<image src="/icons/home.svg" class="icon" />
```
